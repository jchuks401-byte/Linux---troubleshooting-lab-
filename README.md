# Linux Troubleshooting Lab

## 📌 Overview
This project demonstrates basic Linux troubleshooting skills including process management, file permissions, and software installation.

## 🛠️ Skills Used
- Linux commands (ps, kill, chmod, sudo, apt)
- Troubleshooting techniques
- Process management
- Permission handling

## 🔍 Scenarios

### 1. Process Management
- Started a background process using `sleep`
- Identified process using `ps aux`
- Terminated process using `kill`

### 2. File Permissions
- Encountered "Permission denied" error
- Resolved using `chmod +x`

### 3. Software Installation
- Encountered permission issue
- Resolved using `sudo apt install`

## ✅ Outcome
Gained hands-on experience troubleshooting real Linux system issues and applying appropriate fixes.
